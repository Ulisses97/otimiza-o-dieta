# Instalação do projeto

Instalando o Python:
sudo apt-get install python3


Instalando o PIP:
sudo apt-get install python3-pip


Instalando o PuLP:
pip install pulp


Instalando o Pandas:
pip install pandas


Instalando o Xlrd:
pip install xlrd

Instalando o django
pip install django

# Rodar servidor
python3 manage.py runserver
